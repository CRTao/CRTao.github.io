前言

大三時修習圖形識別概論時，認識了MLP與NN架構的觀念，

但課程上主要使用 MATLAB 來做處理，用裡面現成的函式庫也不需要處理程式環境。

現在做專題需要用到 python 來做處理，

雖然可以一層一層刻，但是有鑑於時間與勞力的分配問題，

還是要交給機器學習開源函式庫：Tensorflow 與 深度學習框架 Keras 啦。

當然也有其他的選擇，這邊就用這兩者為基礎。

安裝

  1. 安裝 python

python 一定要是”64-bit”的版本，不然安裝tensorflow時會出錯。

害我又得重裝一次python，不知道有沒有漏裝什麼需要的函式庫。

  1. 架設 虛擬環境

我參考的用書是使用 Anaconda 來架設虛擬環境。

也有 python 預設的 virtualenv。

但我是用 pipenv 的方式來建立虛擬環境。

也不知道之後會不會遇到其他問題。

詳細可參考：
Pipenv: Python Development Workflow for Humans

  1. 安裝 Tensorflow 與 Keras
pipenv install tensorflow
pipenv install keras
  1. 運行 虛擬環境

啟動環境

pipenv shell

關閉環境

( directory_folder-XXXXXXXX ) > exit

全部完成後，前置環境應該就沒問題了。

該是手動建立測資的時間了。

每組樣本至少1000張吧。

真麻煩。
